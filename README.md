# IPL Cricket Analytics (2008–2025)

## Project Overview
This project presents an end-to-end analysis of Indian Premier League (IPL) cricket data using **Python, SQL, and Power BI**.  
The objective is to derive meaningful insights at **Match, Team, and Player** levels through data cleaning, querying, and interactive dashboards.

---

## Tools & Technologies
- **Python** (Pandas, NumPy) – Data cleaning and preprocessing  
- **SQL** – Analytical queries and aggregations  
- **Power BI** – Interactive dashboards and visual analytics  
- **Excel** – Query result validation and exports  

---

## Project Workflow
1. Raw IPL match and delivery data was cleaned and transformed using Python  
2. SQL queries were written to analyze match, team, and player performance  
3. Interactive dashboards were built in Power BI with KPIs, slicers, tooltips, and charts

## Match Analysis – Key Insights
- IPL has been played across **36 venues** over **18 seasons**, with a total of **1169 matches**.
- A cumulative **374K runs** were scored, indicating a high-scoring tournament overall.
- The tournament recorded **14K fours** and **14K sixes**, highlighting the dominance of boundary hitting.
- Total wickets across all seasons stand at **14K**, showing a balanced contest between bat and ball.
- Teams choose to **field first in nearly 65% of matches**, emphasizing the advantage of chasing in T20 cricket.
- Recent seasons show an **upward trend in total runs and wickets**, reflecting more aggressive gameplay.
